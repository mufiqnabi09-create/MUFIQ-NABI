Project Overview

This project predicts the likelihood of customer arrivals during different times of the day using frequency-based probability analysis in Microsoft Excel.

Objectives
Analyze customer visit patterns.
Calculate customer arrival probabilities.
Identify peak and slow business hours.
Develop an interactive Excel dashboard.
Support data-driven business decisions.
